AI Video Editor - Integración con iFrame

Instrucciones:

Para incrustar esta aplicación en tu propio sitio web:

  1. Reemplaza http://localhost:5000 con la URL donde está alojada tu aplicación.
  2. Asegúrate de que la aplicación tenga configurada correctamente las opciones:
    • enableXsrfProtection = false
    • enableCORS = false
  3. Ajusta el tamaño del iframe según las necesidades de tu sitio web.
